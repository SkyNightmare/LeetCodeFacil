# LeetCodeFacil 💻

**Website:** [https://leetcodefacil.netlify.app/]  

**Description:**  
LeetCodeFacil is a blog where I share step-by-step explanations and code solutions for LeetCode problems. The purpose of the blog is to help people who are starting their coding journey and want to improve their problem-solving skills. The website is in **Spanish**.  

Built with **Astro** and **Tailwind CSS**, it provides a clear and structured way to learn coding through examples and explanations.  

---

## Key Features

- **Step by Step** — Focus on **C# code** with detailed, step-by-step explanations.  
- **Most Popular LeetCodes** — Covers the most common LeetCode problems with ready-to-use solutions.  

---

## Technologies Used

- **Astro**  
- **Tailwind CSS**  
- **C#** (for code examples)
